Smart Guard System - Lock Module

Project for Centennial College's course

Implemented with C++ and Arduino Nano 33 IoT board. Working with difference sensors and output components to achieve the automation and data collection. The Arduino nano 33 IoT is the core of the lock module, it will receive the signal from the magnetic door switch, the push button, and the RFID reader. Then, it will control the RGB LED, stepper motor and buzzer based on the input. Moreover, it receives the request from admin portal and provides the response.
